# Multiplayer Arcade Server

Backend server for the Multiplayer Arcade game platform using Socket.io.

## Setup

1. Install dependencies:
```bash
cd server
npm install
```

2. Start the server:
```bash
npm run dev
```

The server will run on port 8000.

## Features

- Real-time multiplayer game synchronization
- Room-based game sessions (up to 4 players per room)
- URL-based room joining
- Automatic room cleanup for empty rooms
