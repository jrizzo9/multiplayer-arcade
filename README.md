# Multiplayer Arcade

A multiplayer arcade game platform built with React and Tailwind CSS.

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

The game will be available at `http://localhost:3000`

## How to Play

- Click anywhere or press **SPACE** to make the bird jump
- Navigate through the pipes without hitting them
- Your score increases each time you pass through a pipe
- Try to get the highest score possible!

## Controls

- **SPACE** or **Click**: Jump/Start/Restart
